# youthbit_tg_bot

##backend (work in progress)

нужен postgreSQL и nodeJS
заходим в директорию
<cd ./backend>
`<npm i> `
`<npm start>`

##frontend (work in progress)
Нужна nodeJS

<cd ./frontend>
`<npm i> `
`<npm run start>`
или если старая версия nodejs
`<npm run start-node> `


##Бот
Нужна nodeJS 

`<cd ./server>`
`<npm run start-bot>
`
